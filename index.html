3. 想法

本博客主要用于记录一些技术讨论和生活随笔。

4. 如何使用

新建文章：hexo new “文章标题”
删除文章：Blog\source_posts目录下直接删除对应的md文件。
部署、发布：hexo g，然后 hexo d
更换主题：在Blog目录下git clone 主题git，然后修改_config.yml文件中的theme。
